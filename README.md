#  Flappy Bird
**Project Matakuliah Proyek Game**

Flappy Bird adalah game arcade 2D di mana pemain mengendalikan burung kecil
yang harus melewati celah antar pipa tanpa menabrak. Setiap pipa yang dilewati menambah skor. Game berakhir saat burung menabrak pipa.

---

##  Informasi Game

| | |
|---|---|
| **Nama Game** | Flappy Bird |
| **Genre** | Arcade 2D |
| **Mode** | Single Player |
| **Kontrol** | `Space` untuk loncat, Mouse untuk menu |
| **Sistem Skor** | Real-time |
| **Game Engine** | Godot 4.0 |
| **Bahasa** | GDScript |
| **Platform** | Windows, Linux |

---

##  Fitur

-  Kontrol simpel — `Space` untuk loncat, mouse untuk menu
-  Pipa muncul secara acak dengan celah bervariasi
-  Sistem skor real-time
-  Layar **Game Over** dengan skor akhir
-  Menu **Settings** — atur volume BGM & SFX
-  Musik latar (BGM) dan efek suara (SFX)

---

##  Cara Bermain

| Aksi | Input |
|---|---|
| Loncat / Flap | `Space` |
| Akses Menu Settings | Mouse (klik) |
| Restart | Klik tombol di layar Game Over |

##  Download & Cara Menjalankan

Pilih platform sesuai sistem operasi, lalu download:

-  [Download Windows](https://github.com/shitodcy/flappybird/releases/download/beta-version/windows-version.zip)
-  [Download Linux](https://github.com/shitodcy/flappybird/releases/download/beta-version/linux-version.zip)

Setelah download:

1. Extract file `.zip` yang sudah didownload
2. Jalankan file di dalam folder hasil extract:
   - **Windows** — klik dua kali `FlappyBird.exe`
   - **Linux** — jalankan `FlappyBird.x86_64` via terminal atau klik dua kali
